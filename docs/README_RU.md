# SacramentuOS 0.3.5

SacramentuOS — учебная консольная система. Основной режим запуска для современных ПК: `flash/EFI/BOOT/BOOTX64.EFI` с FAT32-флешки.

В версии 0.3.5:

- `cls` заменён на `clean`;
- `ver` показывает только название ОС, версию, дату сборки и дату последнего обновления;
- `color` теперь поддерживает `color list` и выбор по номеру: `color 1`, `color 10`;
- удалены лишние команды `set`, `get`, `env`, `unset`, `grep`, `head`, `wc`, `sleep` из UEFI и Legacy;
- README переписан на русском и английском с актуальным списком команд.

Главная команда для справки внутри системы:

```text
help
```

Главная структура для флешки:

```text
flash/EFI/BOOT/BOOTX64.EFI
```
