SacramentuOS changelog
======================

0.2.3 Compact Help IDE Keys
----------------------------
- help/docs стали компактнее: меньше пустого места и лишних вложенных разделов.
- commands теперь выводит команды плотной сеткой.
- Старый строковый редактор заменён на полноэкранный RAM-редактор.
- В редакторе работают стрелки, Home, End, Enter, Backspace и Delete.
- F7 сохраняет, F8 сохраняет и выходит, F9 выходит без сохранения.
- Ctrl+C отменяет ввод в shell, выход из редактора и команду sleep.
- Лимиты редактора увеличены до 8 файлов, 64 строк и 119 символов на строку.

0.2.2 Green Console Editor
--------------------------
- Экран очищается перед большим ASCII-логотипом SacramentuOS.
- Цвет по умолчанию изменен с голубого на зеленый.
- Команда color теперь меняет accent color для prompt, логотипа и документации, а не только текущий текст.
- Добавлен выбор username при загрузке.
- Prompt теперь имеет формат username@sacramentuOS>.
- Команда whoami показывает выбранного пользователя.
- set USER <name> меняет username и prompt во время работы.
- help без аргументов теперь открывает встроенную документацию.
- Добавлена команда docs.
- Убраны фановые команды из command list: dice, coin, fortune, matrix, rand, banner, logo, box, progress.
- Добавлен безопасный RAM code editor: edit, files, show, rm.
- grep/head/wc теперь работают и с RAM-файлами редактора.

0.2.1 Boot Fix
--------------
- Добавлен запуск ядра напрямую через QEMU: make run-kernel.
- Добавлена цель make run-kernel-built.
- Обновлены инструкции для Windows 11 / WSL.

0.2.0 Extended Console
----------------------
- Расширен shell.
- Добавлены RAM notes/todo, env, виртуальные файлы и системные команды.
