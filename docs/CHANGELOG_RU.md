# Changelog

## 0.3.5

- `cls` заменён на `clean` в UEFI и Legacy.
- Исправлен `ver`: теперь без лишней подписи UEFI Edition.
- В `ver` добавлены название ОС, версия, дата последней сборки и дата последнего обновления.
- `color` получил режим `color list` и выбор цвета по номеру `0..15`.
- Удалены лишние команды `set`, `get`, `env`, `unset`, `grep`, `head`, `wc`, `sleep` из UEFI и Legacy.
- Обновлены встроенные `help/docs`, README, flash-файлы и Makefile.

## 0.3.2

- Добавлен автоматический выбор лучшего UEFI text mode при старте.
- Уменьшено мерцание редактора.
- Обновлена структура проекта: `flash/`, `docs/`, `scripts/`.

## 0.3.1

- Добавлена полноценная структура для UEFI-флешки.

## 0.3.0

- Добавлен нативный UEFI-режим `BOOTX64.EFI`.
